# 循环

<!-- 讲解标签 blocks break -->

在 zig 中，循环分为两种，一种是 `while`，一种是 `for`。

Zig 还允许我们将 `while` 循环作为表达式使用，此时需要搭配 `else` 和 `break` 语句。
